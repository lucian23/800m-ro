# 800m.ro

Site static pentru proba de 800 m atletism, publicabil prin GitHub Pages pe domeniul `800m.ro`.

## Structura

- `index.html` - continutul paginii
- `styles.css` - design responsive
- `app.js` - calculatorul de ritm, splituri, proiectii si sesiuni
- `CNAME` - domeniul custom `800m.ro`

## Continut inclus

- hero si navigatie pentru site-ul public
- calculator de splituri pentru 800 m
- proiectii pentru 400 m, 600 m si 1000 m
- sugestii de antrenament pe baza timpului tinta
- model de cursa pe segmente
- biblioteca de sesiuni
- trepte orientative de performanta
- sectiune pregatita pentru resurse si articole

## Actualizari viitoare

Site-ul poate creste cu articole, exemple de microcicluri, analize de curse, interviuri, rezultate si bareme oficiale.
