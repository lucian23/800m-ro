# 800m.ro

Site static pentru proba de 800 m atletism, publicabil prin GitHub Pages pe domeniul `800m.ro`.

## Scop

`800m.ro` este gandit ca o resursa practica in limba romana pentru sportivi, parinti si antrenori. Site-ul explica proba de 800 m prin instrumente si repere usor de folosit: splituri, ritmuri, tactica, antrenamente, teste si planificare saptamanala.

## Structura

- `index.html` - continutul paginii si sectiunile principale
- `styles.css` - design responsive, layout si componente vizuale
- `app.js` - calculatorul de ritm, planificatorul si filtrarea bibliotecii
- `CNAME` - domeniul custom `800m.ro`

## Functionalitati incluse

- hero modern si navigatie pentru site-ul public
- calculator de splituri pentru 800 m cu profil de cursa
- proiectii pentru 400 m, 600 m si 1000 m
- recomandare de sedinta in functie de obiectivul ales
- planificator saptamanal dupa nivel, perioada si numar de sedinte
- model tactic pe segmente: 0-200, 200-400, 400-600, 600-800
- biblioteca de antrenamente filtrabila: viteza, specific, aerobic, tehnica
- teste si repere de monitorizare
- tabel orientativ de performanta
- principii utile pentru pregatirea probei de 800 m

## Publicare

Repo-ul este pregatit pentru GitHub Pages. `CNAME` pastreaza domeniul `800m.ro`.
