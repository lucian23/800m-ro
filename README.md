# 800m.ro

Site static pentru proba de 800 m atletism, publicabil prin GitHub Pages.

## Structura

- `index.html` - continutul paginii
- `styles.css` - design responsive
- `app.js` - calculatorul de ritm si splituri
- `CNAME` - domeniul custom `800m.ro`

## Actualizari

Site-ul este gandit pentru update-uri regulate: articole, exemple de antrenament, analize de cursa si resurse pentru atleti/antrenori.
